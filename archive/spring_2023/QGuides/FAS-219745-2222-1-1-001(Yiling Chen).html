What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
This class is not easy and in many ways the first few weeks don't feel like an applied math class. There is a lot of theory to work through before getting to the applications, and even then the applications remain pretty surface level compared to AM121. Unlike other courses, there aren't a ton of resources to draw on if you are struggling. I'd really only recommend this class if you loved AM121 and are ok with a more complex version of it.
This is a very challenging course, and you should take it if you have a strong understanding of material from AM121 and things from linear algebra. This course does take up a lot of time to understand, but once you have mastered the initial elements of the class, things start falling into place. I felt that my foundation for this class wasn't strong enough, so this made me assignments at the end of the semester become difficult. Also, go to office hours if your experience in programming is not that strong, since you will need help for that in problem sets.
You learn a lot in this class, but it's also really challenging, so it might be best to take it only if you're comfortable with proofs. Yiling Chen is very knowledgeable and a good lecturer. The textbook is really good and although the exercises are hard, they teach you a lot.
The course is not as difficult as I imagined. The in–class practice questions are helpful to understand the key concepts in convex optimization. For some problems you need to calculate a lot of derivatives though...
