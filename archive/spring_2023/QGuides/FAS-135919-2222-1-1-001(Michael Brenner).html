Professor Brenner is awesome, and the class is centered around a very interesting topic. Some ML background would be very helpful for problem sets and projects. This course is excellent. Professor Brenner is one of my favorite lecturers/professors at Harvard. He has really good taste in problems and I found every lecture to be super interesting and new. The best part is also that you just get a really good sense of what is going on in ML research applied to different fields and it's such a satisfying feeling to have tried a bunch of novel methods by the end of the semester. It is most rewarding if you take it with prior exposure to ML and/or data science, but in general the course is super accessible and there is a lot of material to learn from. It's structured in a way that what you get out of it is up to you.
The class has really thought provoking psets. Take this if you like using Tensorflow, because you get to practice a lot. If you’re comfortable with it, the course will not be that hard for you. Project components of the course required some work for mini, and a lot of work for main project. Brenner is a really cool teacher with awesome ideas and intuition. Only drawback is that lectures could have been more organized/clearly directed.
should already have prior ML experience
Brenner is an amazing professor and super chill! Also really passionate about the material
Michael Brenner is seriously the best professor. He motivated the importance of developing robust methods for solving inverse problems very well in this class. We developed a pretty comprehensive toolset during the class for tackling these types of problems, and I think this is a really good class if you want to expand your knowledge of computational tools that can be useful in a research settings. If you have taken more than one ML class or done ML research before, a lot of time is spent on these methods, so you might not get so much out of the class if you already know a lot.
This class will really helped me to understand and implement many deep learning methods. Michael is a really nice guy and a great lecturer. Nonetheless, I would not recommend taking this class a semester when you are very busy or don't have a lot of flexibility in your schedule because the lack of organization makes it hard to plan ahead.

This is a great course to take if you want to learn how to do machine learning. They teach you just enough background to understand generally why the algorithm works, and then they teach you how to use packages to solve actual problems. There are 5 psets in the first 6–7 weeks, and then you do a miniproject and a final project. Your code doesn't have to be brilliant to get a good grade––you just need to demonstrate critical thinking and you'll be fine.
This is an excellent class, provided that you have a background in machine learning and python already. There are many things to take away from the course and the guest lectures are always amazing.
The teaching staff are very generous and really don't focus on grades, which was refreshing. That said, the class was pretty unorganized and had a lot of work in the first half, so be prepared for that (but Professor Brenner is very nice and listens to student needs).
Take this class if you want to become familiar with machine learning tools and popular machine learning algorithms that can be applied to problems in the sciences.
Prof. Brenner is a very caring teacher and his class is very interesting for stem researchers who want to use DL!
This class sounds super cool, and that's because the material is exciting and seems useful. This is true, but the course is also super difficult. A single problem set might take you weeks to solve if you are learning the material for the first time. Before taking this course, I strongly recommend looking through the syllabus to gauge your familiarity with the material. If you have not worked with CNNs, PCA, gradient descent, and other standard applied math tools, you will find this course very challenging and might want to consider a different course.
This is a very solid applied machine learning course at Harvard that I think is a great follow–up to CS181, and possibly a good alternative to CS109ab. The course staff are amazing, and Prof. Brenner is an excellent instructor who cares a lot about his students. A few parts of this course may be review, but you learn so much about the cutting–edge, applied side of machine learning (as opposed to the more theoretical CS181) that it is definitely worth it. The problem sets are also very eye–opening and worth the time you spend on them.
Definitely take this class! It's a decent amount of work if you're not super familiar with python and ML, but you learn a lot.
