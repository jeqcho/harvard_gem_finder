SO MUCH FUN!! This class was definitely the highlight of my week. There were lots of opportunities to practice public speaking in both formal and informal settings, and I made so many new friends from this class :)
I will say, though, that I didn't find this class to be as "life–changing" as I've heard people call it in the past. In terms of public speaking techniques, I don't think I learned too much that I didn't already know (like the importance of using logos, ethos, pathos, etc), but this class definitely provided a great environment to practice all these skills.
