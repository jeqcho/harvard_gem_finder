What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
I would take this class if you care about really learning how to use data science in a concrete useful way that doesn't waste anyone's time with busywork and problem sets. Andrew is a great professor who is truly kind, caring, helpful, and encouraging.
Would recommend if you have a project you're really interested in building or exploring. This class will give you the time and space to help you do it. I wished this class had more of an emphasis on teaching us new models and code, but there wasn't really much of that. Overall, the class lets you work on a project you want in a laid–back, supportive environment. We didn't get any grades throughout the semester, so I can't really speak on what grading is like. Would not recommend this class if you are not proficient in Python or R (only need one of these), as there is no coding taught in the class.
In this current iteration, the majority of focus is on the completion of one data science project. If you want to follow one project from start to finish, this is the course for you. If you are looking for a traditional classroom experience, this is not for you.
For this class, it wouldn't be bad to come in with an idea of what you'd like to research/create/do since the class is basically a large structured project. This will obviously adapt as you hear and learn things, but for many their ideas were improved upon as opposed to completely changed. Overall, it's less a methods class and more a project class that teaches you the process of data science – at least in this iteration.
This course gives you a ton of freedom to work on your project. This also means there is nothing stopping you from putting off your project until the very last minute. DO NOT DO THIS YOU WILL REGRET IT. The course staff provides incredible feedback and direction but at the end of the day you need to work on your project in order to do well in the course.
