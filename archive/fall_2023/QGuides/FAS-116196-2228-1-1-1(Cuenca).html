This course was difficult but rewarding! The material was really well chosen, and if you're interested in algebraic combinatorics, this course would be interesting for you. It was not the best–organized class I've taken, but Cesar tends to be available for questions and is a really great lecturer.
This is a very fun class. The workload is not too heavy and the content is very accessible. If you enjoy problems that you can easily sink your teeth into, I think you'll like this class. Cuenca is a very clear lecturer—though the lectures a bit slowly sometimes—and is very helpful in answering questions. The problem sets were all quite interesting and very doable.
