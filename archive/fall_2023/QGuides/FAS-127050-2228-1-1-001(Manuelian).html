TAKE THIS COURSE! Total diamond in the rough, hidden among the anthropology courses. This course literally provides the opportunity to immerse yourself in the world of Giza. The professor, TF, and all the resources in the course are just excellent, and are rather "exclusive" to the course, so every lecture makes it feel like you're receiving new and cutting–edge content.
This is a wonderful class, the content is really interesting and the classroom is very cool. I would make sure to keep up with your readings and presentation reviews, but otherwise, as long as you're an active participant in class, you should find this course rewarding.

If you’re interested, take it. It doesn’t require any specific background, so if you want to do it you will be able to! I loved the class, the professor & TF were so enthusiastic about their field and so so helpful. The workload is reading–heavy but reasonable, although the end of the semester gets a bit more intense with the final project, although that’s definitely doable if you keep up with it.
This class has some really interesting subject material and Professor der Manuelian is awesome!! There are a lot of required readings so be prepared for that. Section can be boring but does allow you to start work on your project early.
I highly recommend this class if you are looking for something new. I took this class to push my slightly out of my comfort zone and really enjoyed the material. Some topics in the course can get slightly repetitive so I would recommend reading up on the topics and deciding if you are interesting before you sign up for this class. Also, the work definitely begins to pick up towards the end of the semester, especially with the final project so take this class seriously and it will be incredibly rewarding.
