Comments
If you're hoping to learn specific writing techniques (e.g. grammar, syntax, flow), then this probably isn't the class for you. However, if you are hoping to learn about overarching principles of scientific writing—such as how to make a timeline for writing a journal publication, how to make clear figures, and how to determine authorship—then this class covers all that and more. Suzanne and Dan really care about this class and put in so much effort to make it an enriching experience for everyone, and it's a great class to develop confidence in your overall writing abilities.
I think this is a great class to take early–on in your academic career, especially if you're a Ph.D. student. It will give you a new perspective on writing and your relationship to it as an academic. No matter what, you will come out of this class as a better writer (even if you already think you're decent at it).
Fun course, classes are long but enjoyable, assignments are very manageable, you get out of the class what you put in, profs are very enthusiastic

A great course if you care about writing and want to improve your scientific writing skills.
This is an amazing class with really good instructors! The discussions were really fun and I always got a lot out of them. I highly recommend this course to all grad students.
