I really enjoyed this class, but it is a weird one because the first unit is all bio and chem stuff (digestive system, etc.) so any background in that helps enormously so it doesn't feel so crazy difficult. After that first unit it is a lot more accessible to anyone, and definitely fascinating to learn about so many facets of food.
I took this class randomly because it looked interesting– IT WAS. Professor Warinner is actually amazing. She’s so cool, and approachable and really tries to help everyone succeed. The readings can get pretty tough because they tend to be long and we have to do them before every class. There are also pop quizzes that cover details of the readings. I would definitely recommend taking this class because you will probably never learn in such detail how we have evolved and adapted to food and the food politics of America. As a freshman, this class had a lot of upperclassmen and I would recommend taking expos and having some knowledge of bio/chem before taking this class. This course can be a lot of work, but it’s worth it in my opinion.
Def take this class. Very wide scope of things learned and readings can pile up, but the information is so interesting!
There is a lot of reading. I liked that this class straddles the line between archaeology and historical/current events, which makes it interesting to both archaeology majors and non–archaeology majors.
Take this class ONLY if you are interested in the material. Pop quizzes are really difficult ( coming from someone who did the readings and took notes on them). But the tests are fair. Know that this class has a final 10pg essay and a final exam which can be really challenging. Also the class covers a lot of material, but I found the course to be really interesting and had a lot of ramifications to today.
Although the material is very interesting, there is a lot of reading. The pop quizzes are super annoying so if you're taking the class for a letter grade I would recommend a reading group. I would also recommend reading smart. The pop quizzes aren't hard. When it comes to articles read the introduction, read the conclusion, and skim smart (something I regret not doing). In terms of books, you could break up the reading. I think the readings are very interesting but very hard to do especially if you have other reading–heavy classes. I saw "GEM" in other course evaluations and I would have to disagree. Readings are super important later in the class when it becomes more discussion focused or else class is awkward. The sections were super fun and hands on. Overall, if you're taking it for fun like I was, I would recommend to take it pass fail (my biggest regret). Also the first midterm is very memorization focused. I would also recommend you answer the questions as you go or find a group to answer them as you go because if not the study guide becomes quite unmanageable. The final is a little easier if you've gone to most or all the lectures and have done most or at least half of the readings. Despite its difficulty for me, I'm glad I took this class. You learn sooo much. This class is also moderately DAO accommodation friendly. I would recommend communicating your needs throughout the semester (although communication can be hard). Also remember to ask for extensions early if you need them. Also do the extra credit throughout the semester. Don't miss class because you can't make up the pop quizzes unless you have a good reason such as a doctor's note. Also keep in mind some weeks it's less work while other weeks when you have assignments such as a project, the midterm, the final paper, and the final it's a LOT more work.

It may well change the way you see food.
