What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
If you have already taken 120a/b, 130a is definitely a continuation with a similar structure in homeworks, dictations, review tests, etc. The real difference is that 130 is graded much more harshly, and it feels like there is much more material to learn each week. But, if you have already made it this far in the Chinese department and genuinely enjoy learning the language, I still highly recommend taking this course.
KEEP TAKING CHINESE! It is always the most fun, best taught class of the semester for me and you gain a really cool community through your classmates. I will never stop recommending this department to people I know.
If you have taken any other Harvard Chinese classes, the format of this one is exactly the same with essentially the same assignments, test format, teaching style, etc. I definitely feel like my level has improved quite a bit since the beginning of the semester. This year they did not make us learn how to write characters, which was very helpful because it allows you to put more effort into actually internalizing the vocabulary and grammar.
This class is a step up in work load from Chinese 120. The structure of the class is essentially the same but you learn more vocab and move faster. Just try to keep up with the vocab as much as you can or you'll be forced to cram it all in before the review tests and exams.
