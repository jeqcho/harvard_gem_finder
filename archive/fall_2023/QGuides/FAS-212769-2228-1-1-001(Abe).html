Very easy class but you do learn a lot
Professor Abe is the sweetest/kindest person. I know he pours so much into this class and it is very evident. TAKE THIS SEMINAR. If you are interested in learning about Buddhism, reading sutras and having a small class of people just like you, then you wont be disappointed. It is also a very low stress class. The readings are extensive sometimes but not too overwhelming and not all necessary to spend hours trying to understand because you talk about it during class. I loved this class so much. Will be sad to leave it!!
It is what you make of it. There are A LOT of readings, and if you do all of them, you will definitely learn a lot. Regardless, I think everyone in the class walked away with a much greater understanding of Buddhism than what we started with. It was a bit boring at times, and I think the course could be improved with more interactive things like more meditation sessions, but it was still very interesting. The best part of this class was definitely Prof. Abe, everyone loves him. Definitely take this class if you want to learn more about Buddhism, and take a chill class with a great prof.
