It's a great class. Very fun and you learn many cool things. It's a really interesting topic and the teacher teaches it well
This class starts off a little slow and you might be a little confused about why a particle physicist is teaching you are astrology, but by the end of the course all of the topics that are covered really come together and it is cool to see the narrative that is created by moving through time and discovering the history of scientific understanding and then the physics itself.
The class is very heavily focused on less purely scientific subjects at the beginning and only becomes physics heavy for the last few classes. If you are hoping to take a fully physics freshman seminar, this is not the one for you, though that is not to say we did not learn a LOT of physics at the end.
I would tell future students that this class explores fascinating topics and you receive such a wealth of knowledge that makes the class enjoyable.
If you like learning about everything space related from astrology to string theory, this class is for you! If you only like some space related things, you're going to have some weeks where you really dislike the class, so I would only really recommend it if you have interest in both the humanities subjects like astrology and like the divine comedy and learning about astrophysics.
