I really enjoyed this course! You'll need to know what's going on so not exactly a class you can slack off in, BUT there are a lot of resources for you to do well! I highly recommend going to office hours because that's where I really got to understand the topics. The take–home tests were great and felt pretty low–stakes, which was super nice. The fact that the problem sets were graded off neatness was super nice, too, because then you didn't feel so much pressure to get every single thing right, and the emphasis was more on learning the material!!
Definitely recommend having a basic foundation in genetics (like not going in completely blind), but with the right prep, this was a really low–stress overview of a lot of topics in genetics. The exams are take–home and the homework is graded primarily on completion, so I was never stressed about my grade and could focus on enjoying the course material.
If you are interested in learning more about genetics, this is a good class. I'm not sure this is the best class for someone who wants to just take an OEB class and I think if the format of the class was improved a little the class would be more engaging.
