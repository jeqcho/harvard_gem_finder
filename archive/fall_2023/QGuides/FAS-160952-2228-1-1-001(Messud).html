Comments
This class was the highlight of my semester! Even as someone who has been challenging myself to improve as a writer and as a reader since I can remember, this class felt like a “leveling–up” experience for how I read, write, and exist in community with other writers.
This class is amazing! Professor Messud is wonderful. She is supportive, organized, and provides extremely helpful feedback. I cannot recommend this class enough.
Treasure every moment!
This class is a wonderful creative writing seminar. It has a very optimal layout for both learning and writing (half exercises and normal reading discussion, half workshops). Its size makes for an intimate environment that lends itself nicely to workshops. You also get a sense of each student before you start hearing their feedback on your work, which is extremely helpful. I would highly recommend this for anyone interested in developing as a writer, or anyone who wants a class that will give them a scheduled time to write admit a busy semester.
This class is extraordinary–– even if you haven't taken a creative writing workshop before, Claire will teach you everything you need to grow in such a wonderful, supportive, and boundary–pushing environment. You will read phenomenal works of literature, and learn about how character works with all the other elements of story–telling to create spell–binding narratives. So awesome. Take this class if you can!
Professor Messud's courses are terrific fun! You will be challenged to write and read creatively and critically, and you will have a lot of fun creating your own work and reading your classmates' work. Workshop courses do require commitment to giving feedback and setting aside time for writing, so make sure you are enthusiastic/ready to give this commitment!
