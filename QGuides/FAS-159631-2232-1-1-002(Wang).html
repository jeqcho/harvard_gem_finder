What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
Only take this class if you are really committed to Chinese, and you have the time to spend 10+ hours a week on it. This class is rewarding, but very fast paced with a high volume of new words and structures each week. It can be very hard to keep up in class if you do not review before, so making sure you have the time to review vocabulary before each lecture is very important. Overall, the teaching staff make this class great, but it was still one of the harder classes in my semester.
Continuation of the Chinese series, not too different from the previous classes like 130a. Not too difficult if you put the work in, and fairly rewarding.
If you have made it this far in the Chinese department, then you will definitely like this class. It is a direct continuation Chinese 130a. There is a lot of emphasis on grammar.
