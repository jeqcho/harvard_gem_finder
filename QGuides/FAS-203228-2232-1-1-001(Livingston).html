This is a really good class if you want to advance in Latin to be able to start reading a little more independently. It's the first move away from learning Latin as only through grammar, to just being read Latin as any other language. What I'm trying to say is that things like language comprehension and being able to "read" rather than just "translate" become a lot more important. I think my class was kind of unique in that we used tiered texts(three difficulty levels for each text, and you try to go through all three successively) rather than the normal, just get a singular passage of the original text and attack it. I definitely prefer the tiered texts, it made the aforementioned transition a lot easier, but there is a pretty steep learning curve in the beginning. Said learning curve is pretty standard though, and definitely more alleviated through this method.
