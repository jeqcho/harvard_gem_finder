The class presented a lot of challenging material in a fairly accessible way and was a really good overview of deep learning – especially given the rapid advances in the field. While the discussion–based format was not ideal, the lectures and discussions still had a lot of interesting concepts.
Melissa is just the BEST. She is insanely––INSANELY––smart but also incredibly humble about her academic prowess. She makes complicated matters in deep learning seem so easy. It is so enjoyable to be in her class and learn from her. Take this class, it is absolutely essential for people to learn about NLP and CV and other novel methods in deep learning. Easy to say it is the best class i've taken at Harvard.
This class is a great course for those interested in practical research applications in economics/history for primarily document imaging data. It doesn't go into the super technical details, but is very much is grounded in the technical details. Python and technical AI/ML experience is basically necessary for this course and generally is almost all grad students. Go to office hours and talk with the Professor, it was a very enriching course!
Definitely take it! Really at the frontier, you'll learn many techniques and work on a fun project. I don't think you need to be fully fluent in pyhton to take it.
This class is good at providing a general overview of modern methods in machine learning/the data analysis pipeline. If you want a lot of specifics, though, this class may not be for you. I was not a fan of the flipped classroom format, and I think a lot of people ended up skipping class for most of the semester since class time was not super informative. I found the implementation exercises to be most helpful in applying what we were learning about. Tom was an amazing TF and was extremely helpful.
