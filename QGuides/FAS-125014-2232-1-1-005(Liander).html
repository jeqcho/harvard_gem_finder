This is such a great class!!!! If you're a heritage speaker (like I am,) this is a nice course that really helped me clear up some previous confusion with certain grammar rules. We're assigned a movie each week to watch, longer assignments include 2–3 page essays. Overall it's super super manageable and all of the content was really interesting! I had Ricardo as a TF and he's amazing!!! So fun and really helps facilitate a sense of community within the classroom.

The only thing thing is is that if you feel that you already have decent Spanish, you might want to consider skipping ahead and asking if you're able to join a higher–level course. I don't regret taking this class at all, but I only realized afterwards that I could've taken some other courses (like Spanish and the Community) if I had just known to ask beforehand. (Useful if you want to get a citation and also want to focus more on a specific subject!)
