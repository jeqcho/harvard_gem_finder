You could get by if you really wanted to if you didn't read the books, but this is a class where you get out what you put in–the syllabus is incredible. Take this class if you like reading but also want a gem!

This is a flawed but excellent course. Fisher has a great reputation, which is mostly deserved. His lectures are excellent. The syllabus is mostly excellent, although I wish that there were more works by POC. But know that there is a ton of reading and Fisher literally never gives extensions. In fact if you ask for an extension he will tell the entire lecture that no extensions will be granted, including for religious reasons. The workload is doable so it's not extremely unreasonable, but it's important to know going in. If you're only planning on taking one English class at Harvard, this is a good contender.
