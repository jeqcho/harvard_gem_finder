What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
This is a very interesting class especially to Americans who want to be exposed to a more critical view of Economics. The structure of the class allows for a great amount of autonomous study and exploration, and the topic is very fascinating. The professor's classes are very well structured.
This is basically a watered–down law school course. Unger speaks in high prose and you aren't expected to understand every word. As long as you keep up with readings, the class should be understandable. Would recommend cooperating with the other undergrads in the class as there will be very few. Would definitely recommend.
As someone who had no background or knowledge about the political economy, this course was so interesting! You will learn about the transformation of the knowledge economy: how we changed from an agrarian–based economy to one that is dominant and driven by knowledge. The readings are helpful in helping you understand the lectures. Make sure to read the readings before class to give you background. Since the readings can be dense, breaking them up (not reading them in one sitting) is very useful. Overall, if you want to really learn about the current political–economic climate, this is a must–take course that merges history, economics, policymaking, and law! The course is very theoretical in nature, but assignments are structured in an applied nature.
