***TAKE THIS CLASS! The homework is light, and don't be intimidated by the in–class compositions. With ample preparation, the compositions are easy and graded very leniently in the first round of grading. The class is very laid–back but is discussion–based so do participate when possible. If you're looking to grow as a French speaker and writer, this class is for you.
I loved this class! If Amanda Gann teaches a section, I highly recommend taking the class with her!
Good French class and good topic. The section is pretty fun and engaging but we all mostly talked to the two closest people to us so it could be more engaging. The only hesitation I would have before taking this class is that it is a major time commitment, especially compared to the other French classes. I think there is just so much busy work that I was not expecting, which made me feel like I shouldn't have taken the course... Consider that before enrolling but otherwise, it's a good class and Amanda was a great TF.
Be sure to prepare well for the in class writings and vocab quizzes. Also, do the dossier sections when they're assigned so you don't fall behind.
This class is a fair amount of work (dossiers, exchange conversations, compositions) but as long as you start early/stay on top of everything, it's not too unmanageable. Prof. Gann was super nice and a great teacher and also great conversation facilitator. A lot of the topics the class goes into too are pretty interesting, and sometimes the class feels like a seminar which is cool. Best part of this class is definitely the lack of exams. I'd definitely recommend if you have the time or if you want to take an interesting French class.
I recommend the class! The work is not too bad as long as you stay on top of it. You learn a lot about different political and social topics, so if you're interested in that, this class will be great.
