Please take this class if you are in any way interested in Statistics, even if you are not focusing on Biostats. Professor Parmigiani is a true expert and chill guy. Working with gene data is super rewarding and beating the benchmark in the final project has to be one of my best learning experiences at Harvard. Even if you are not interested in Biology, the analyses you learn in this class have a lot of functionality beyond the field.
This class is amazing! Professor Parmigiani really cares about his students and wants them to have engaging discussion around the course material. There is also lots of support for assignments, and there is not really any super high pressure to be perfect. The structure of the course forms a community. Participating in the class regularly is very rewarding, and I felt myself learning a lot of practical knowledge about how to operate in this field with biomarkers. I wish I could take a course like this every year, I think it was one of my favorite ones at Harvard that I've ever taken. The main message: excellent professor and TF, not super high stress assignments, and you still learn a lot.
Don't bother if you are not a stat concentrator or need it for a requirement. To really understand the material, you need a lot of 110 and 111 knowledge. However, I had none and did fine because the problem sets were easy (only because the TF gave us code in section). If he had not done that, this class would've been a nightmare. Definitely easier work wise than any stat class I've taken.
Interesting material and great professor. Generally light work–load compared to some other stat classes. Solid opportunity to gain some more experience in R.
This class is a great class for learning statistics is actually applied to real medical problems like cancer prediction.
Interesting application of statistics to real–world biological data! This was also the first time I used RJAGS/implemented MCMC simulations. I like how Prof. Parmigiani shares insights from his experience working with different statistical models during lecture; you don't find little tips like these in textbooks. It's also fun how Prof. Parmigiani engages us in the classroom by asking us to imagine we're the FDA and a pharma company comes up to us with the following data/model, etc. Chill, low–stress course, though I think it's best if you've taken Stat 110, 111, and 139; it is assumed that you're familiar with statistical models and distributions (Gamma, Chi–Squared, Normal, hierarchical models, etc.).
Only criticisms: I feel like topics could've been presented more in depth and more connected to each other. Also, I found that homework was very easy if you went to section or reviewed the section materials, but impossible if you go on lecture alone – not sure if this is good or bad.
The projects really make this class. The lectures are a little hard to follow because the notation is shifty; there's almost no math at all in the course; and the psets are mostly copying provided code and modifying it and interpreting the outputs. However, the three projects – predicting some outcome from some set of data – are quite interesting, and they really let you explore or create statistical techniques. Project presentations feel like lab meetings except everyone actually knows what's going on and can provide good feedback. All in all, not a hard course but interesting if you're content with an intuitive explanation of what's happening statistically. (Also, it's geared towards juniors and seniors, but you can definitely take it as a sophomore if you have the pre–reqs done.)
