Chem 154 is a great class if you enjoyed Chem 40 and are looking to learn more about the *why* to everything you learned in Chem 40. The details aren't skipped and you learn the math behind everything. Prof. Nocera is a good lecturer, and you will definitely learn a lot in this class! Problem sets can be tricky but my TF was always willing to help and exams were pretty straight forward. Definitely would recommend if you are interested in inorganic chem!
You should take this class if you love inorganic chemistry
Great course, gives excellent context for how certain transition metal complexes behave spectroscopically
A very enjoyable class. Having some math background (especially in group theory) is helpful.
This is the best inorganic class I've taken. You get to learn about the math and underlying principles behind many inorganic concepts that you just memorize the first time you take undergraduate inorganic chemistry. For example, you get to learn how character tables are constructed.
