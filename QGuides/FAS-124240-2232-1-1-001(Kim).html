What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
If you enjoyed Korean BA, definitely take BB! It's more of the same except better :)
The course is generally pretty rigorous (class every day Mon–Fri and very frequent quizzes) but it's honestly totally feasible if you buy into it. As long as I kept up with the assignments and quizzes, I actually didn't need to study much on my own time because it's so well managed and thorough on a daily basis.
if you come in knowing no Korean (like me!) then this class will be so so hard. it was easily the hardest class I've ever taken at Harvard. Half the class already knows the language.

That said, the support system for this class is so great. the instructors are all amazing and while the class was incredibly difficult I have nothing negative to say. be ready for the difficulty but if you are excited about Korean, take it!
always pay attention in class because you could get called on at any moment and you don't want to be embarrassed when you don't know the answer
