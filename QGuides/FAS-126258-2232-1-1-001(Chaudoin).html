I think if students are looking for a deep dive into "international conflict and cooperation," this isn't necessarily the right class for you. It combines simple econ theories with politics and then has you apply them to some real–world events, but basically in class we talk about theories like game theory and prisoner's dilemma instead of different international conflicts around the world and the motivations behind them. Because of this very simplified way of thinking about politics, all the assignments in the class become a checklist of what you need to say, instead of really being able to apply the concepts in complex ways. I wouldn't recommend this class if you're looking for a deeper dive into the actual history and complexity behind international conflicts, but only if you want to learn about how some intro econ models apply to politics. Also, the two lectures of class were easily simplified into a few slides in section. I think the content of the course isn't enough to be a course and so it really tries to stretch at times and ends up being somewhat dull/easy to zone out in.

Overall, I think the way the class was set up detracted from the actual learning of the student (focuses very harshly on minute and I would say sometimes irrelevant details), so I wouldn't recommend this class if you want a comprehensive learning experience about international conflict and cooperation.
Future students should be aware that this class is very formulaic in nature, applying theories and mathematical systems of representation to real world scenarios. While to some students, this is the crucial to understanding and predicting world events, to other students it can come off as a reductive understanding of world affairs. Aside from that, this is a brilliant course that everyone should take in order to learn the theory to apply as you wish. The teaching is incredible and super supportive with accessible resources and space to request help as needed. The content is very engaging with much to learn from the class.
