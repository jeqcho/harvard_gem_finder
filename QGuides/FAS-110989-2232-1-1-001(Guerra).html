This class is not for everyone. If you are "afraid" to take 22a, just take the leap. This class is way too computational and has almost no reasoning component... It's more like pattern matching and rarely you would have to come up with something new. The psets start getting really boring and you will become tired of doing the same thing over and over again. I would not take this class again, but I have learned quite a lot of stuff. Unlike most people say, exams were definitely not hard, just again – computationally intensive. Be ready for this class. The work just cumulatively builds up and never really gets lighter (unlike for Math 21a). Therefore, make sure you're taking some easier classes on the side.

Lastly, I would say definitely do practice tests when studying for exams; they really help (are almost essential) when studying for tests.
