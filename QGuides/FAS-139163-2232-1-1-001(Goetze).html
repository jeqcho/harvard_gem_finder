The course materials were cool, but class was unengaging and the lecturer brought in several guests (and did not show up for those sessions). In return, I think students reciprocated.
A philosophy class suitable for concentrators and students from technical backgrounds. Professor is great at directing interesting discussions so the 2 hours goes by quickly. Final project is equivalent to a sophomore/junior tutorial
This is an excellent course, highly recommended for those interested in tech and AI, and should be made mandatory for those doing CS and those working in data sicence.
Trystan is a wonderful instructor. The course has many consistent weekly assignments (blog, QFC, and milestones for the project), so there's a lot to keep track of. The final project was a great opportunity to explore an area of interest, and I'm grateful for the flexibility that Trystan gave us. Guest lecturers were helpful and informative too. Overall, a good course.
If you're looking for an intro to tech ethics, this class is good for you. If you already have somewhat of a background, it might not be the most useful class. The readings are quite interesting and are selected very well, but the class component tended to be a bit dry. The class is probably half a Gov class and half a Phil class.
If you're interested in "ethics of cs/technology" then take this class! Beyond the weekly readings and blog posts and discussions which make you think more about each area of possible danger, the research project that you do allows you to really take ownership of a problem that you're passionate about.
The class is wonderful. Readings and assignments are very interesting and manageable. The course is mostly discussion based, and Prof. Goetze does a great job guiding the discussion so it never feels boring. The final project is spaced out over multiple weeks so it's not just a ton of work at the end. The course is structured so that you're always making progress. The professor is also very accommodating if you need to move a deadline or get help.
