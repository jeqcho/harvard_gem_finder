This class is AMAZING! Brian is a wonderful teacher. If you have an interest in biology or music...this class if for you! Or even if you aren't...this class is still great. You don't need any previous experience in these topics. Every class we watch videos, listen to sounds, and discuss. Honestly this class is a 10/10...I highly recommend it.
