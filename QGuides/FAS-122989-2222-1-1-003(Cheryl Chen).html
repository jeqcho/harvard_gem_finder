Take this class! Cat is fantastic, and doing philosophy in a group setting is infinitely richer.
This class was in my top few favorites at Harvard (and I'm a senior, so that's saying a lot!). The topic is fascinating, and Cat presents it in a super accessible way, so no worries if you aren't a philosophy or technology expert! But also, if you are an expert, you will still learn a lot. Cat structured this course in a way to really help you improve on your writing, and I found this really really helpful. I also loved how the assignments were structured so that they encouraged participation in the course, but didn't feel like busy–work, and also were evenly distributed throughout so that no one assignment felt like it would make or break your grade. Cat created an incredibly supportive and fun and stimulating learning and classroom environment. I actually knew everyone's names and we would always say hi to each other when we saw each other across the street! Unfortunately, that kind of classroom–based connection is so rare on this campus, but Cat did an incredible job facilitating it. I cannot recommend this class enough!
