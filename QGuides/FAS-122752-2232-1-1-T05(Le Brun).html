Pick a good course head and do not worry about the topic
I took Professor Rice’s class on Applied Health Economics. He was super laid back, which made our class a really nice environment because everyone felt comfortable asking a ton of questions that went behind the reading. He was always fair on grading.
Love the data science aspect of this class
Start the paper early!! Finding datasets for the final paper can be challenging – sometimes you need to request for data and that can take a while, so make sure to budget sufficient time. Also take advantage of R and Stata office hours, there are lots of those!

Personally, I decided to learn Stata through the coding psets, only to realize at the very end that I do not like using Stata and reverted back to using R, which I was more comfortable with, for the final project instead. I think there could potentially be value in exposing yourself to a new coding language, especially since Stata is widely used in economics, but I would not recommend doing what I did because it made the final project slightly more painful since I had to relearn the econometric techniques in R (also the coding psets because I was raging all the time at Stata haha).
