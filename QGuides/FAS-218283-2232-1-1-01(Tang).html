This has been one of my favorite classes that I've taken at Harvard. Kwok–leong is one of the funniest and most down–to–earth professors I have ever had in this college. The course is great because it teaches you how to use Chat GPT to build a lot of interesting projects. However, I do think that if you've never written a line of code before that this course will be quite difficult. I think this is an excellent introduction, however, on how to use Chat GPT and tools in the digital humanities to build really cool projects that you can use to showcase on your resume as well! Please take it!
