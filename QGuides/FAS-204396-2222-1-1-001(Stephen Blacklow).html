It is very focused on proteins and drugs, but it is an overall great class for a basic understanding of protein biology.
Be patient and focus on concepts
This is a pretty nice, relatively easy class that exposes you to a lot of biochemistry research around HMS. It is a bit heavier on the drug discovery side than I expected from the course description. The frontiers lectures are the heart of the course, and with those, you get out what you put in. If you're really interested in one of the papers, you can do a lot more background reading and talk to the frontiers lecturer and you'll learn a lot. If you don't really care that much/aren't interested in the paper, you can also phone it in and be find grade–wise. It's really up to you.
Good class, though not as heavy on protein mechanics like the title of the course suggests.
Don't take it if it's not required or if you aren't interested in proteins/kinetics/etc.
This is an incredibly well organized class with a moderate workload. If you're interested in protein biophysics or physical chemistry, this would be a great class for you!
More homework than any class I have taken here, but definitely interesting topics that can be useful to learn.
This is a fantastic class for a budding young protein scientist or chemical biologist. Take this class if you want to learn about protein binding interactions, methods for studying protein biochemistry, and drug developmental approaches/screens.
