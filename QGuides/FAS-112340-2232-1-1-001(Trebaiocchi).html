What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Comments
This is a great class, and if you'd like to learn a lot of Italian quickly, I'd highly recommend it. Be aware, it is a significant time commitment – but it is quite enjoyable.

The class size is very small, so you are able (and need) to be very engaged. I found this very helpful for learning a language. There is homework every night, which is something to get used to versus a typical class with maybe weekly problem sets or the like.

Prof Lieberman was great – very engaging, passionate, and kind.

If you're debating between Italian 15 and Italian 10, I'd say, shop them both, and see how you feel. But if you want an intense Italian language experience, this class is hard to beat!
it is super easy to get behind in this course if you miss a few assignments or classes. you learn a lot in a very short period of time, which is nice, but having to make every class with no option to make them up asynchronously or anything was very difficult for me as a student with health issues/disabled student.
If you're interested in Romance Languages and speak another one fluently, take this class! Professor Lieberman is an incredible teacher. 2023–2024 will be his last year able to lecture here, so if you're interested in all in taking ITAL–15, take it this year!!!! He's taught it for like 7 years or something, and there will be a new, probably less experienced professor when he leaves. His selection of course materials is so great (you'll leave with a nice repertoire of Italian songs, movies, music videos, poems, and readings). The assignments are varied enough to be interesting and engaging but structured enough to not be too difficult. The section component is the heart of the course and you'll enjoy going, even though 5 days a week is a lot. You'll put a lot of work in but get a lot out of it. You'll be able to get around Italy by the end of the course; it's truly remarkable.
The only downside to the course is that you have to pay $150 for course materials. I hope the Italian department fronts this in future classes!
This class is A LOT of work, and you will only get out what you put in. There is homework every night and it is extremely easy to fall behind. You will learn a lot of Italian, but you will spend a lot of time studying in and out of class. That being said, there is no better class to learn a lot in a short amount of time.
