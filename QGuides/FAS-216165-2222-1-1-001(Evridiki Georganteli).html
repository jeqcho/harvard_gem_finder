I really wish I could give this course a higher rating. The teaching staff, especially Professor Georganteli, were so amazing and dedicated. As an example, Professor Georganteli put in the effort to learn everyone's name in a ~70 person class. The section component was also so unique and cool . Each week we would go to a Harvard museum and have special access to artifacts relevant to what we were studying each week. We handled most of these objects with our bare hands such a wonderful opportunity.

The core issue with the course is that it was designed for maybe 25 students max, but ~70 were actually enrolled. The main example of how this became an issue was in how student presentations happened. For the second half of the course, there was a student presentation almost every class session. These were supposed to take the first 10 minutes of class with the rest reserved for regular lecture. That would work if each group were 2–3 people, but with a class this size, the groups ended up being 6–8 people and taking up the **entire class session.** There's a time and a place for student presentations, but having them dominate the second half of the course made class time pretty much a wash. A smaller way this class size issue played out was in course 'debates.' With a group of ~70 students, these were less debates and more a string of people one after another saying disconnected things for participation points. Some other misc issues: sections were a highlight of the course but could have used more structure. Also, if you miss a single course session you have to write a 600–word reflection on the readings by that Sunday. If you're out sick (say, with COVID) for a week and miss 3 classes between lecture and section...do the math.

I think this course has a ton of potential, but only if enrollment is capped or if the teaching staff embraces the realities of teaching a larger lecture course. As it's currently managed, about 30% of the course is super wonderful and memorable, and the other 70% is...also there.
