What would you like to tell future students about this class? (Your response to this question may be published anonymously.)

Table for [QTitle].
Comments
Overall, I think this course provides a satisfactory second introductory class to Spanish. The topics presented are interesting and the class focus on conversation and communication allows students to grow more comfortable speaking the language conversationally. My main criticisms of the class would be that sometimes I felt like class time would be better spent teaching the topic content from the beginning, instead of having a page in the Canvas module about a concept and then having little time to learn how to apply it. I think the class experience would have been better if some classes were saved for lectures or reviews. Also, I feel like the class experience and grading is highly TF dependent, which can make or break one's experience, so there needs to be further standardization or occasional check–ins from the department of how class is supposed to be run, what content/concept has been covered, what materials are given to students before the test, etc, how homework and tests should be graded, etc.
Make sure to study the canvas modules pretty well because they will be the best way to make sure you have the necessary tools to succeed in the class. Paying full attention and studying them like class work will set you up very well.
